    Class CommentRangeStart

    Represents the start of a comment range in a WordprocessingML document.

    Marks the beginning of a region of text that is associated with a comment. Must be paired with a CommentRangeEnd with the same ID.

    Reference: http://officeopenxml.com/WPrun.php
